Block Structures

There are three basic block structures that you can build from. Examine how the structures work in Firebug, Dragonfly, or by applying the stylesheet mod_debug.css

mod

Mod is the basic container, all other containers inherit from this one. This is the high performance container that should be used unless you specifically need another type of container.

AG

AG should be used in cases where you require images for borders because borders or drop shadows are too complex to be simulated via borders on mod and/or inner.

pop

Body - use for popups and other containers that need outside transparency.
